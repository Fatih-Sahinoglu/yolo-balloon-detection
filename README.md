# yolo-balloon-detection

This repository contains a YOLO-based balloon detection system developed
for educational and experimental purposes.

## Features
- Real-time balloon detection using YOLO
- Region of Interest (ROI) based scanning
- Center point and coordinate calculation
- Confidence score visualization
- Nearest target detection and line drawing

## Notes
> Trained YOLO weights (.pt files) are NOT included in this repository.  
> Datasets used during training are not included.

## Requirements
- Python 3.9+
- OpenCV
- Ultralytics YOLO
